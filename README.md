# 협업 테스트용 프로젝트

### 실행 방법:
1. git clone한 저장소로 이동
2. CLI에서 ``make`` 혹은 ``mingw32-make`` 를 입력
3. 그러면 각 파일의 변경 사항이 'testprog.exe'라는 파일로 한번에 컴파일됨.
4. 그거 실행하면 됨.

----

1. main.cpp를 만들어서 거기 wndks(), tjsrb(), dudxo() 함수를 넣음(그냥 한영타 반전임)
2. 각 함수는 void. 그냥 printf로 자기 이름 출력하는 거
3. 각 함수는 함수 이름과 동일한 이름의 .cpp 이름의 파일에 기록되어 있음

**목표:** 컴파일된 파일을 실행했을 때 우리가 printf로 넣은 게 정상적으로 출력되어야 함
